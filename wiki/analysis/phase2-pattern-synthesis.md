## 1. The Coercive Control Architecture

The 15 RF patterns do not operate in isolation. They form a **four-layer coercive control system**:

```
Layer 1 — CONTROL THE CHANNEL
  RF-02 (Gatekeeping) + RF-03 (Weaponized Logistics) + RF-08 (Triangulation)
  → All Eli-Olivia contact is mediated through Melissa.
  → Olivia is the enforcer by Phase B.

Layer 2 — DESTROY THE IDENTITY
  RF-01 (Alienation) + RF-04 (DARVO) + RF-15 (False Allegations) + RF-09 (Gaslighting)
  → Eli is characterized as inadequate, predatory, dangerous, and unstable.
  → Each new incident is amplified; each allegation is denied by Eli = used as evidence against him.

Layer 3 — MAINTAIN COMPLIANCE
  RF-05 (Legal Threats) + RF-06 (Financial Coercion) + RF-07 (Boundary Violations)
  → 26 legal threats create a compliance environment without requiring execution.
  → Financial threats reinforce behavioral modification.

Layer 4 — INSTRUMENT THE CHILD
  RF-08 (Coaching) + RF-11 (Child Involvement) + RF-01 (Preference Relay)
  → Olivia is coached to relay negative content, enforce the intermediary rule, and provide access-denial authority.
  → By Phase C, Olivia's stated preferences are the stated justification for every visit refusal.
```

**Key interaction — the feedback loop:** Layer 1 (control the channel) ensures Layer 2 (identity destruction) reaches Olivia without Eli's rebuttal. Layer 4 (instrument the child) converts Layer 2 content into Olivia's own "organic" preferences. Those preferences are then relayed back through Layer 1 to Eli as Layer 3 compliance pressure. The system is self-reinforcing.
